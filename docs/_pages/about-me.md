안녕하세요! 👋

**이진재**입니다. 2년 2개월차 개발자로, 다양한 기술 스택을 활용한 프로젝트 경험을 쌓아오고 있습니다.

## 나는 누구인가

Flutter, Spring Boot, Next.js, React 등 다양한 기술을 다루며, 새로운 기술을 배우는 것을 즐깁니다. 처음 보는 기술이라도 빠르게 적응하고 실무에 적용하는 것에 자신이 있습니다.

## 내가 해온 것들

### 모바일 앱 개발
Flutter와 Spring Boot를 활용해 기존 PHP 기반 앱을 전면 재구현한 경험이 있습니다. Firebase를 활용한 소셜 로그인, 실시간 채팅, 푸시 알림 기능을 구현하고, 구글 플레이스토어와 애플 앱스토어에 성공적으로 배포했습니다.

### 웹 개발
Next.js와 TypeScript로 반응형 랜딩페이지를 처음부터 개발했습니다. 리소스 최적화와 AWS 배포를 통해 높은 성능과 사용자 경험을 제공했습니다.

### 백엔드 개발
Spring Boot와 JPA를 활용한 REST API 개발, 복잡한 쿼리 최적화, 데이터 표준화 및 자동화 작업을 수행했습니다.

## 내가 좋아하는 것

코딩을 하면서 느낀 점은, 같은 기능이라도 누가 구현하느냐에 따라 완전히 다른 코드가 나온다는 것입니다. 언어마다 다른 방법으로 문제를 해결하는 과정을 알아가는 것이 정말 흥미롭습니다.

새로운 기술을 배우고, 여러 사람들의 의견을 듣고, 제 방식으로 해석하며 풀어나가는 과정을 좋아합니다. 그래서 더 많은 기회와 경험을 쌓고 싶어 개발자로 일하고 있습니다.

## 나의 강점

**빠른 학습 능력**: Flutter를 처음 접했을 때, 강의와 도서 학습은 물론 GDG 컨퍼런스에도 직접 참여하며 빠르게 역량을 쌓았습니다.

**주도적인 문제 해결**: 신규 프로젝트부터 기존 시스템 리팩토링까지, 주도적으로 문제를 해결하며 책임감을 가지고 임합니다.

**실무 경험**: 앱을 성공적으로 스토어에 출시하고, 실제 사용자들이 서비스를 활용하는 모습을 보며 성취감을 느꼈습니다.

## 앞으로의 계획

최신 기술 트렌드를 빠르게 습득하고 실무에 적용하는 능력을 계속 발전시켜 나가고 싶습니다. 사용자에게 가치 있는 서비스를 만드는 개발자가 되고 싶습니다.
